ecryptfs is not the package name on Ubuntu Noble — the correct package is:

ecryptfs-utils

Install it like this:

sudo apt install -y ecryptfs-utils

mkdir -p ~/vault
sudo mount -t ecryptfs ~/vault ~/vault


sudo umount ~/vault


or use

./crypt.sh
